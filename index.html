MBoutique

MBoutique is a full e-commerce website dedicated to selling macarons. This application features the LAMP stack. AngularJS was used as the front-end MVVC to retrieve and manage customer and shopping data on the client. PHP was usedto interface with the MySQL database to send information to client and implement a token login system. The entire site is hosted on an Apache server on an Unbuntu (Linux) machine hosted by AWS EC2. This application utilizes HTML5, CSS3, Bootstrap, Material Design Lite, Angular JS, Google Maps API, PHP, OAUTH Token System, and MySQL.

Password Manager

The Password Manager is a simple implementation of a password manager utilizing Node.js. The password manager is used via the command line or terminal, and allows the user to create, read, update, and delete their accounts as long as they have a master password. The accounts are stored locallly in an encrypted file. This application primarily uses Node.js and various modules from npm, like yargs and crypto.js. This was also my first time using Ecmascript 6 syntax, since the current version of Node.js allows developers to code in ES6 syntax.

Node.js Chat

Node.js Chat is a simple chat application that utilizes socket.io. This aplication allows the user to create their own custom chat rooms on the fly. Users are notified when a new user joins and when a user leaves. This application also features a custom command that allows all users within a room to be displayed to the user that requested the information. This application features many of the npm modules that make programming in Node.js a pleasure, like express.js, moment,js, and socket.io
