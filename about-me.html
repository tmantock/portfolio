I Specialize In

HTML5

Hypertext Markup Language 5. The latest an greatest version of the web's most popular markup language. Web sites would be nothing without the skeleton that brings it all together. With new features like video, canvas, and SVG, HTML5 has proven to be a strong technology worthy of its dominance in the web industry.

CSS3

Cascading Style Sheets 3. The latest and greatest version of the web's most popular style sheet language. With new features like animations, box-sizing, columns, and multiple backgrounds, CSS3 continues to prove why CSS is the dominate language for styling markup languages.

Bootstrap

Bootstrap 3, the Twitter built front-end framework, provides to tools for making a website that responsive across multiple devices like, desktops, tablets, and smartphones. Bootstrap makes styling webpages with uniformity and responsiveness in mind simple.

Javascript

Javascript. The most popular language of the web. Javascript is easy to debug, allows for object oriented design, DOM interaction, and allows for asynchronous communications. These powerful features make Javascript.jQuery the world's most utilized programming language to increase graphical user interfaces and interactivity of the front-end.
